favicon&og 퍼블리싱 가이드

1. Favicon

  1. favicon에 사용되는 img 파일은 ico 또는 png
  2. ico 파일 사용시에는 link rel에 "shortcut" 추가
  3. 16x16: 브라우저용 / 32x32: 작업표시줄 단축키용 / 48x48: 윈도우즈 사이트 아이콘 /
    96x96: 데스크탑 단축키용 / 180x180: 애플 터치용
  4. 최소한 준비해야할 사이즈는 16x16, 32x32
  5. png 파일의 경우에는 필요한 sizes를 반드시 선언한다

2. Open Gragh

  1. <meta property="og:type" content="website(기본값)">
  2. <meta property="og:url" content="사이트 표준 url">
  3. <meta property="og:site_name" content="사이트 대표 이름">
  4. <meta property="og:title" content="콘텐츠 제목">
  5. <meta property="og:image" content="사이트 대표 로고(1.9:1 비율 이미지 권장)">
  6. <meta property="og:description" content="사이트 설명">
